Spicy Green Book

Web application

Spicy Green Book is a nonprofit organization that compiles a directory of Black owned businesses and performs marketing services for them of charge.

I volunteer my time as a web developer assiting with frontend development. The application is built with React Native, Expo, and Next.js and is hosted/deployed on Vercel. Because my knowledge is limited in that space, I contribute where I can while actively learning the technologies.
